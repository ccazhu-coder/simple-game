# 簡單遊戲 - Simple Game

一個用 Python 和 Pygame 製作的簡單遊戲示例。

## 遊戲說明

- **目標**：控制綠色方塊躲避紅色敵人
- **控制**：使用左右箭頭鍵移動
- **計分**：每躲避一個敵人得 10 分

## 安裝與運行

### 前置需求
- Python 3.8+
- Pygame

### 安裝依賴

```bash
pip install -r requirements.txt
```

### 運行遊戲

```bash
python main.py
```
