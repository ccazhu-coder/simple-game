快速開始

跟著下面的步驟，5 分鐘內開始遊戲：

  1. 安裝 Python
    訪問 python.org 下載 Python 3.8 或更新版本
  2. 創建專案資料夾
    在您的電腦上創建一個資料夾來存放遊戲
  3. 下載遊戲文件
    git clone https://github.com/ccazhu-coder/simple-game.git
  4. 安裝依賴
    cd simple-game
    pip install -r requirements.txt
  5. 運行遊戲
    python main.py

遊戲說明

🎮

玩家

綠色方塊 - 您控制的角色

👾

敵人

紅色方塊 - 要避開的敵人

目標：使用 左箭頭 和 右箭頭 鍵移動，躲避紅色敵人。每成功躲避一個敵人得 10 分！

系統要求

  • Python 3.8 或更新版本
  • Pygame 2.5.2
  • Windows、macOS 或 Linux
  • 至少 100MB 硬碟空間
